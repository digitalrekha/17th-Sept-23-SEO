Definition of Sementic Elements

Semantic elements are the elements that provide more meaningful descriptions of the content of a web page. These elements help web browsers and search engines understand the structure and content of our page, which can improve the way our website is displayed and indexed.

Types of Sementic Elements

  1. <header>: The main heading of the page
  2. <nav>: Links to other pages or sections of the website
  3. <section>: A group of related content
  4. <article>: A self-contained piece of content
  5. <aside>: Additional content that is not essential to the main content of the page
